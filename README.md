# Zen Finance

ZenFinance is a full-stack personal finance management application that helps users track accounts, income, expenses, and transactions with a clean and secure experience.

Built with Spring Boot and React, ZenFinance focuses on JWT-based authentication, efficient data handling with MongoDB, and a minimal, intuitive UI.

## Key Features

- JWT authentication with BCrypt password hashing
- Dashboard with total balance and monthly income/expense summary
- Multiple account management
- Income & expense transaction tracking
- Profile and settings management

## Tech Stack

- Backend: Java, Spring Boot, Spring Security
- Frontend: React, Tailwind CSS, Vite

## Purpose
ZenFinance is a resume-grade full-stack project showcasing secure backend design, REST APIs, and a modern React frontend.
